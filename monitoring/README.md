# Production-ready система сбора и визуализации метрик
Создать полноценную систему мониторинга, которая собирает метрики со всех уровней инфраструктуры: от железа до приложений, визуализирует их в дашбордах и отправляет алерты при проблемах. Система должна быть отказоустойчивой, масштабируемой и готовой к использованию в продакшене.
