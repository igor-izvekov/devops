# Система автоматического аудита и инвентаризации серверов (Server Audit & Inventory System)
### Концепция проекта
Разработать инструмент командной строки для автоматического сбора, анализа и отчетности о состоянии серверов. Это будет полноценная утилита, которая:
- Собирает информацию о системе (hardware, OS, сеть, процессы)
- Проверяет безопасность (user audit, открытые порты, подозрительные процессы)
- Генерирует отчеты в разных форматах
- Умеет работать удаленно по SSH
- Имеет режим мониторинга (запуск в фоне)
