# Database DevOps: CI/CD для управления схемами данных как сделать этот pet проект

## Концепция проекта
Суть проекта: Автоматизировать процесс внесения изменений в схему базы данных (PostgreSQL/MySQL) с использованием подхода GitOps. Каждое изменение схемы проходит через Git-репозиторий, автоматическую проверку (линтинг, проверка обратной совместимости) и контролируемое развертывание в различных окружениях (dev/stage/prod).
